I‚Äôd like to do one more example with the two rigid boundaries before moving on ‚Äì mostly because in this example there‚Äôs an interesting twist with how we‚Äôll have to solve the partial differential equation.

2.2.4 Self-Similar Flows

The techniques we used to solve the three previous problems ‚Äì integrating an ordinary differential equation, using separation of variables to solve a partial differential equation, and then doing the same thing but with the added twist of using a steady state solution to handle nonhomogeneous boundary conditions ‚Äì are all standard techniques physicists use to solve differential equations. All three problems also had very similar set-ups ‚Äì two boundaries and plane parallel shear flow.

I‚Äôd like to do one last example of time-dependent plane parallel shear flow before moving on to other kinds of situations, but this time I‚Äôll remove the boundary at y=h ‚Äì we only have one boundary in this new problem, and it‚Äôs along the x-axis. Fluid exists only in the region y>0, but it can extend up to infinity ‚Äì see Figure 2.4.

As with the previous example, this boundary will be impulsively put into motion at t=0 with a velocity (to the right) of U. We‚Äôve already gone through the arguments in the last section about what kind of flow this is, and they all hold here, too ‚Äì the only difference is in the boundary conditions. That means we once again have the partial differential equation

‚àÇ‚Å°u‚àÇ‚Å°t=ŒΩ‚Å¢‚àÇ2‚Å°u‚àÇ‚Å°y2. (2.28)

Now, we could, if we wanted, go through the same procedure as before ‚Äì find a steady-state solution, apply separation of variables, and build our final answer as a superposition of solutions. Consider, first, some dimensional analysis. Examining Equations 2.28-2.31 we can see that our problem depends, at best, on four variables: U, ŒΩ, y, and t. But if we write the velocity as a non-dimensional variable,

Okay, where are we going with this? Well, since f must be dimensionless, and since it can only depend on ŒΩ, y, and t and no other quantity, we need to rearrange those three variables into a combination that is dimensionless. It turns out there is only one way to do this, the combination

This whole idea ‚Äì having the solution depend only on a dimensionless quantity ‚Äì is called self-similarity, for reasons we‚Äôll see in a bit.
